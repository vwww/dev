Gameplay

If the ball hits the ground on your side, you lose the round.

To move your player, use the mouse, touchscreen, or WASD/arrow keys.

Future features

This project is still under development. These features are planned:

  • Smarter (unbeatable or variable-difficulty) bots
  • Client-side Prediction (extrapolation), possibly only sending state after collisions

Options

Serve

A random player serves in the first round. Who serves in later rounds?

  • alternate (default): the player who did not serve last round
  • winner: the winner of the last round
  • loser: the loser of the last round
  • random: random

Intermission Time / s (default: 1, min: 0, max: 3)

Time between rounds after a point is scored.

Player Size (default: 1, min: 0.1, max: 4)

The player size is scaled by this factor.

Ball Size (default: 1, min: 0.1, max: 16)

The ball size is scaled by this factor.

Net Size (default: 1, min: 0, max: 2)

The net size is scaled by this factor.

Game Speed (default: 1, min: 0.25, max: 8)

The game speed is scaled by this factor.

Player Speed (default: 1, min: 0.25, max: 8)

The player speed is scaled by this factor.

Ball Speed (default: 1, min: 0.10, max: 8)

The ball speed is scaled by this factor.

Gravity (default: 1, min: 0.10, max: 8)

The gravity is scaled by this factor.
