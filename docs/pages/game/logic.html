Related

Other people have developed solvers for these types of puzzles.
  • Johannes Singler's LogicalSolver web app (2013), ported from a Java applet (2001)
  • Darra Ricks's Logic-puzzle solver Java app (2013)
  • Bryan Deter's logic-puzzle-solver Ruby library (2008)
  • Michael Benson's Mystery Master site with JavaScript solver (2017?), registration required?
