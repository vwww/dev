Player Types

Human players are controlled in the user interface.

First bot always chooses the lowest-numbered positions, so you can easily beat it.

Random bot chooses whatever it “feels like”, even if it is a bad choice. If you don’t like its choice, you can undo your move and redo it.

Nice bot tries to lose as soon as possible, tie, and then win as late as possible. If it chooses all of its moves under default rules, it never wins.

Optimal bot tries to win as soon as possible, tie, and then lose as late as possible. If it chooses all of its moves under default rules, it never loses.

Table Results

Forcable means the current player can force the desired result. Forced paths are those that allow the current player to keep the result forcable.

Possible means the result is possible, but the opponent can force a different result. Lost paths are those that allow the opponent to force a different result.

Impossible means that the result is not possible.

Result means that the result is guaranteed, as other results are impossible.
