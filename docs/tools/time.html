This page shows time perception according to the model proposed by Lemlich (1975). Under this model, the amount of subjective time that passes per amount of real time is proportional to total elapsed subjective time. When the differential equation is solved, subjective time is proportional to the square root of real time.

Hover in one chart to see the corresponding time in all others.
