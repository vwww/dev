I made a solver for the game Fill (on Google Play and Apple Store) for the stages that have a unique solution (before a certain stage).

It runs in O(nα(n)) time and O(n) space for n cells, but while it fully solves most stages, it cannot fully solve all. By guessing and eliminating possibilities, some additional levels can be solved with O(n²α(n)) time.

Some stages can be solved only partially by inspection, which is what this algorithm does for efficiency. The Hamiltonian path problem for general grid graphs is an NP-Complete problem.
